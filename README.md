Мультибанковское приложение "ВсеФинансы" выполнено для хакатона https://apihack.vtb.ru/ в расках задачи https://apihack.vtb.ru/multibank
Используя множественные api, приложение позволяет агрегировать данные из любого количества банков (базовые - 3, но благодаря гибкой архитектуре можно расшириться)

Среди особенностей приложения: геймификация, позволяющая пользователям получать призы, выполняя банковские операции.

Бэкэнд написан на Java Spring Boot, фронтэнд на JS. В качестве основной БД использованa MongoDB

Есть запушеный пользовательский контейнер, который можно развернуть для ознакомления с принципами работы приложения: https://hub.docker.com/r/chernyukkatrin/multibanking-api (но надо будет развернуть ещё и mongo c redis)
Так же существует полностью собранный контейнер, который достаточно только запустить: https://hub.docker.com/r/chernyukkatrin/multibanking-all-in-one
