Мультибанковское приложение "ВсеФинансы" выполнено для хакатона https://apihack.vtb.ru/ в расках задачи https://apihack.vtb.ru/multibank
Используя множественные api, приложение позволяет агрегировать данные из любого количества банков (базовые - 3, но благодаря гибкой архитектуре можно расщириться)

Среди особенностей приложения: геймификация, позволяющая пользователям получать призы, выполняя банковские операции.

Бэкэнд написан на Java Spring Boot, фронтэнд на JS.
